Suppression des données utilisateur de FindMyPro

Si vous souhaitez supprimer vos données de FindMyPro, suivez ces étapes :

  1. Ouvrez l'application FindMyPro
  2. Allez dans "Paramètres" > "Mon compte"
  3. Cliquez sur "Supprimer mon compte"
  4. Confirmez la suppression

Suppression via Facebook

Alternativement, pour supprimer l'accès via Facebook :

  1. Allez dans les paramètres de votre compte Facebook
  2. Accédez à "Applications et sites web"
  3. Trouvez "FindMyPro"
  4. Cliquez sur "Supprimer"
  5. Confirmez la suppression

Après la suppression, toutes vos données personnelles seront effacées de nos serveurs dans un délai de 30 jours.
